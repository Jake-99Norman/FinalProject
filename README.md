A modern, responive calendar app built with React + TypeScript and Vite.

Users can create, edit, delete events and manage their schecule in a clean nomthly calendar view.

<!-- Features -->

Monthly calendar view
Add new event on any day
Edit existing events
Delete events
Color-coded events
Persistent storage in localStorage
Fully responive layout
Fast performace

https://github.com/Jake-99Norman/FinalProject

https://codesandbox.io/p/github/Jake-99Norman/FinalProject/draft/fast-platform

<!-- How it works -->

The calendar generates days dynamically based on the selected month.

Key components:
Calendar.tsx -renders the monthly grid
DayCell.tsx- individual day square with events
EventModal.tsx -modal for creating/editing/deleting events
OverflowModal.tsx -overflow modal for when events are to many
